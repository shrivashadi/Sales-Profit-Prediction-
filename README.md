## Project Overview
Built a Machine Learning model to predict sales profit using Random Forest Regressor.
Achieved R2 Score of 0.95+ on test data.

## Dataset
- 700 rows of sales data
- Features: Segment, Country, Product, Discount, Units Sold, etc.

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Random Forest)
- Model Saving (pkl file)

## Results
- R2 Score: 0.95+
- Key Finding: Government segment is most profitable
- Key Finding: France is the top performing country
